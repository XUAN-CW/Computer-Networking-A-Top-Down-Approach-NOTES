![2](image/2.png)

打开命令行窗口，输入 **telent**  ,出现以下结果，说明可用。

```
telent
```

## 开始实验

这里用QQ邮箱做实验。打开命令行窗口，输入 **telnet smtp.qq.com 25** ，回车，之后出现如下窗口：

```
telnet smtp.qq.com 25
```
